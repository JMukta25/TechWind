# TechWind

TechWind is a news website inspired from TechCrunch.This website incorporates numbers of news fetched from json database.This website is build using ReactJS framework.

## Run this Application

Clone the repository.
In the project directory,install the dependencies using npm i.
Run the application in the development mode using npm start.

### Walkthrough
TechWind is the news website with collection of informative and detailed news.Entire news can be read by clicking on heading of each news.
In this I have used useState,useEffect,custom hook called as useData and useParams.Also concepts such as props,map are used.
This website is made mobile friendly by using media-query.
SEO using On-pageOptimization in done by add meta-title,description,keyword,error page is designed wih a link to home page,appropriate content fro alt is given.
